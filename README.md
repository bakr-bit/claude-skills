# claude-skills

Private collection of Claude Code skills.

## Install

Drop a skill folder into either:
- `~/.claude/skills/<skill-name>/` (user-level, available everywhere), or
- `<repo>/.claude/skills/<skill-name>/` (project-local).

## Skills

- **wayback-restore** — End-to-end workflow for restoring a dead website from the Wayback Machine into a clean, deployable static site.
